Atuei como líder técnico no projeto de evolução do SiseVida, um sistema desktop legado para gestão de apólices de seguro de vida. O desafio foi redesenhar a arquitetura para um modelo web-based, utilizando ASP.NET e SQL Server, garantindo a integridade dos dados e a continuidade do negócio durante a migração. O projeto melhorou a experiência do usuário e centralizou a operação, reduzindo custos de manutenção.
