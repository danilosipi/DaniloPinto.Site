## Problema

O sistema **SiseVida Web**, da Sistran, apresentava uma interface desatualizada e pouco amigável, o que impactava a experiência de uso dos clientes.  
O desafio era propor um **novo visual mais moderno, responsivo e intuitivo**, sem alterar a lógica funcional já existente no sistema.

## Papel e Responsabilidades

Atuei como **Designer de Interface (UI/UX)**, responsável por:

- Criação da **proposta estética e estrutural** para o novo SiseVida Web.
- **Prototipação navegável** para demonstrar fluxos principais.
- Definição de **paleta de cores, tipografia e padrões visuais**.
- Reorganização da disposição de menus, botões e elementos-chave para melhorar a usabilidade.

## Solução e Estratégia

O trabalho foi focado em um **redesign visual orientado à usabilidade**:

- Layout mais **limpo e responsivo**, alinhado a padrões modernos de UI.
- Uso de cores e ícones para reforçar hierarquia e facilitar navegação.
- Prototipação de telas principais (dashboard, relatórios, cadastros).
- Estrutura preparada para futura implementação por time de desenvolvimento.

## Impacto e Resultados

Apesar de não ter sido colocado em produção, o projeto entregou:

- **Protótipo funcional** navegável, pronto para validação com stakeholders.
- Diretrizes visuais documentadas para orientar evolução futura do sistema.
- Primeiros passos para modernização do **SiseVida Web** em termos de usabilidade.

## Tecnologias / Ferramentas Utilizadas

- **Figma / Adobe XD** para prototipação e criação de interfaces.
- **Ferramentas de design gráfico** para ícones e elementos visuais.

## Destaques

- Criação de **novo layout para sistema legado**, focado em modernização.
- **Protótipo navegável** entregue para stakeholders.
- Padrões de UI/UX definidos para futura implementação.
- Consolidação de proposta estética mais intuitiva e responsiva.
