Liderei o desenvolvimento da plataforma 'EuQueroRevender', um portal B2B que digitalizou o processo de compra para revendedores. O sistema incluía funcionalidades como catálogo de produtos, carrinho de compras, integração com meios de pagamento e um painel administrativo para gestão de pedidos e estoque. O projeto foi desenvolvido com Node.js no backend e React no frontend, entregando uma solução moderna e escalável.
